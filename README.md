# FireMate AI

A professional web interface for an AI-powered tactical decision-support agent. This project demonstrates a smart system designed to assist firefighting forces in the field using NASA satellite data and historical wildfire data.

## Features

- Clean, responsive, and bilingual design (Hebrew & English)
- Interactive AI agent powered by Google Gemini
- Tactical operation orders guided by machine learning algorithm
- Mobile-friendly interface

## Project Structure
